## How to install ?
---------------------

```bash
git clone https://github.com/badcast/darkmoon-kde.git
```

```bash
cd darkmoon-kde
```
   
```bash
chmod +x install.sh
```
    
```bash
./install.sh
```


Set Themes on **systemsettings** in KDE!

Enjoy and Feel!
